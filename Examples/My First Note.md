# My First Note

This is a note. In Obsidian, every `.md` file is a note — a building block of your knowledge system.

What makes Obsidian different from a regular text editor is **connections**. Watch:

- This note is part of the [[Obsidian Synapses]] workshop
- The philosophy behind this system is called [[Concepts/Dracomancy]]
- We're moving beyond the old [[Concepts/The Mind Garden|office model]] into something organic
- This note is a **1D** building block — see [[Concepts/Prompt Dimensions]] for what comes next

Every `[[wikilink]]` you just saw creates a connection in Obsidian's **Graph View**. Press `Ctrl/Cmd + G` to see it. This note is now linked to four other notes. They're linked to others. The graph grows.

---

## Try It Yourself

Create a new note about something you care about. It could be anything — a project, a book, a question, a recipe.

1. Create a new `.md` file (in Obsidian: `Ctrl/Cmd + N`, or ask the agent)
2. Write a few lines about your topic
3. Add at least one `[[wikilink]]` to an existing note — maybe `[[Concepts/Dracomancy]]` if your topic involves transformation, or `[[Concepts/The Mind Garden]]` if it's about how you organize your thinking
4. Check the Graph View — your note is now part of the ecosystem
